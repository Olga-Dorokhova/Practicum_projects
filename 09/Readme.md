# Статистический анализ данных сервиса аренды самокатов GoFast

Чтобы совершать поездки по городу, пользователи сервиса GoFast пользуются мобильным приложением. Сервисом можно пользоваться:
• без подписки
- абонентская плата отсутствует;
- стоимость одной минуты поездки — 8 рублей;
- стоимость старта (начала поездки) — 50 рублей;
• с подпиской Ultra
- абонентская плата — 199 рублей в месяц;
- стоимость одной минуты поездки — 6 рублей;
- стоимость старта — бесплатно.

**Цель исследования:** анализ данных и проверка гипотез, которые могут помочь бизнесу вырасти.

Входные данные: В основных данных есть информация о пользователях, их поездках и подписках.

**ОБЩИЙ ВЫВОД:**

Цель исследования достигнута.

Для достижения цели исследования были решены следующие задачи:
1. Проведена загрузка данных.

В результате загрузки данных мы имеем три датасета:

Пользователи — users
- user_id уникальный идентификатор пользователя
- name имя пользователя
- age возраст
- city город
- subscription_type тип подписки (free, ultra)

Поездки — rides
- user_id уникальный идентификатор пользователя
- distance расстояние, которое пользователь проехал в текущей сессии (в метрах)
- duration продолжительность сессии (в минутах) — время с того момента, как пользователь нажал кнопку «Начать поездку» до момента, как он нажал кнопку «Завершить поездку»
- date дата совершения поездки

Подписки — subsc
- subscription_type тип подписки
- minute_price стоимость одной минуты поездки по данной подписке
- start_ride_price стоимость начала поездки
- subscription_fee стоимость ежемесячного платежа

Наименования столбцов соответствует хорошему стилю.

В данных нет незаполненных строк.

В столбце date датасета rides некорректно отражен тип данных.

Датасеты users и orders имеют общий столбец user_id. Датасеты users и subsc имеют общий столбец subscription_type. Таким образом, в дальнейшем три датасета можно объединить в один.

2. Проведена предобработка данных

В результате предобработки данных:
- был изменен тип данных столбца date датасета rides
- создан новый столбец с номером месяца
- пропуски в данных не обнаружены
- обнаружены и удалены явные дубликаты
- неявные дубликаты не обнаружены.

3. Проведен исследовательский анализ данных.

В результате исследовательского анализа данных определено, что:
- чаще всего сервисом пользуются в Пятигорске, реже всего в Москве
- чаще всего сервисом пользуются пользователи без подписки
- наиболее распространенный возраст пользователей от 22 до 28 лет
- наиболее распространенная дистанция, которую преодолевают пользователи 2,5-3,8км
- чаще всего продолжительность поездки составляет от 14 до 22 минут.

Кроме того были выявлены и удалены аномальные значения в продолжительности поездки (0,52% от данных датасета). Удаленные данные существенно не изменили основные значения исследования.

4. Объединены данные

В результате объединения данных были:
- объединены данные о пользователях, поездках и подписках в один датасет
- созданы два датасета из объединенного датафрейма (с данными о пользователях без подписки и с подпиской)
- выявлены отличия пользователей с подпиской и без подписки

Пользователи без подписки (11тыс.поездок) намного чаще пользуются сервисом, чем с подпиской (около 7тыс.поездок). Существенные отличия в расстоянии и продолжительности поездок:

Для пользователей без подписки:
- наиболее распространенное расстояние для поездки от 2,4км до 3,9км, среднее 3,0км
- наиболее распространенное время для поездки от 13 до 22 минут, среднее 17 минут
Для пользователей с подпиской:
- наиболее распространенное расстояние для поездки от 2,8км до 3,6км среднее 3,1км
- наиболее распространенное время для поездки от 15 до 22 минут, среднее 18 минут

5. Подсчитана выручка

В результате агрегации данных о поездках для каждого пользователя за каждый месяц были:
- найдено суммарное расстояние,
- количество поездок,
- суммарное время поездок,
- посчитана помесячная выручка, которую принес каждый пользователь.

Суммарная выручка от пользователей без подписки значительно выше выручки от пользователей с подпиской. Однако, средняя выручка от пользователей без подписки ниже средней выручки от пользователей с подпиской.

6. Проверены гипотезы

В результате проверки гипотез выяснилось:
- среднее время продолжительности поездок для пользователей с подпиской больше, чем у пользователей без подписки.
- среднее расстояние, которое проезжают пользователи с подпиской за одну поездку, не превышает 3130 метров.
- сpедняя месячная выручка от пользователей с подпиской по месяцам выше, чем выручка от пользователей без подписки.

7. Построены распределения для отдела маркетинга

- Акция по рассылке промокодов - При рассылке 1163 промокодов вероятность того, что менее 100 клиентов продлят подписку, составляет ~4.9%, что удовлетворяет условию задачи (риск невыполнения плана около 5%).
- Оценка вероятности открытия push-уведомления в мобильном приложении - Вероятность 399,500 открытий соответствует 15,37%.

8. Написан общий вывод

**Пользователи с подпиской являются более выгодными клиентами для сервиса аренды самокатов, в связи с этим руководству сервиса необходимо принять меры для перехода клиентов на подписку.**
Те, кто оформил платную подписку, затрачивают больше времени на поездки и приносят больше выручки по месяцам для компании, насмотря на то, что пользователей с подпиской почти в 2 раза меньше, чем пользователей без подписки.

Техническому отделу стоит разобраться с аномальными данными с длительностью поездки 30 секунд (преодолённое расстояние при этом от 4км).
