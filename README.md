# Учебные проекты по анализу данных
Репозиторий содержит коллекцию практических работ по исследованию данных. Проекты охватывают разные предметные области и демонстрируют навыки работы с аналитическим стеком инструментов.

**О репозитории**
Каждый проект — самостоятельное исследование от постановки задачи до выводов. Цель репозитория:
- показать умение работать с реальными данными;
- продемонстрировать навыки анализа, визуализации и интерпретации результатов;
- отразить прогресс в освоении аналитических методик.

Список проектов:
01 - Исследование надёжности заёмщиков

02 - Исследование объявлений о продаже квартир

03 - Исследование данных о российском кинопрокате

04 - Исследование рынка заведений общественного питания Москвы

05 - Построение дашбордов в Tableau

06 - Определение текущего уровня потребительской лояльности

07 - Анализ бизнес‑показателей приложения PROCRASTINATE PRO+

08 - Анализ данных вакансий HH.ru

09 - Статистический анализ данных сервиса аренды самокатов GoFast

10 - Приоритизация гипотез, A/B‑тестирование

11 - Стратегия взаимодействия с клиентами фитнес‑центра

12 - Выявление закономерности успешности компьютерной игры

13 - Анализ оттока клиентов

Структура каждого проекта
— Jupyter‑ноутбуки с кодом анализа;
— презентация (PDF);
— описание задачи, методологии и ключевых выводов.

Инструменты и технологии
Языки: Python (pandas, numpy, matplotlib, seaborn, scikit‑learn), SQL;
Визуализация: Tableau
Эксперименты: A/B‑тестирование, статистические критерии.
