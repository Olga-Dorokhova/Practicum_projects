# Учебные проекты по анализу данных
Репозиторий содержит коллекцию практических работ по исследованию данных. Проекты охватывают разные предметные области и демонстрируют навыки работы с аналитическим стеком инструментов.

**О репозитории**
Каждый проект — самостоятельное исследование от постановки задачи до выводов. Цель репозитория:
- показать умение работать с реальными данными;
- продемонстрировать навыки анализа, визуализации и интерпретации результатов;
- отразить прогресс в освоении аналитических методик.

Структура каждого проекта
- ....ipynb - Jupyter‑ноутбуки с кодом анализа;
- report - презентация (PDF);
- readme - описание задачи, методологии и ключевых выводов.

Список проектов:
01 - Исследование надёжности заёмщиков. Цель проекта - выявить влияние цели кредита, семейного положения, количества детей и уровня дохода клиента на факт погашения кредита в срок. Инструменты: Python (pandas) 

02 - Исследование объявлений о продаже квартир. Цель исследования - провести исследовательский анализ данных и установить параметры, влияющие на цену объектов. Инструменты: Python (pandas, matplotlib, seaborn)

03 - Исследование данных о российском кинопрокате. Цель исследования - исследовать рынок российского кинопроката и выявить текущие тренды. Уделить внимание фильмам, которые получили государственную поддержку. Ответить на вопрос, насколько такие фильмы интересны зрителю. Инструменты: Python (pandas, matplotlib, seaborn)

04 - Исследование рынка заведений общественного питания Москвы. Цель исследования - исследовать рынок заведений общественного питания Москвы, найти интересные особенности и презентовать полученные результаты,
которые в будущем помогут в выборе подходящего инвесторам места. Инструменты: Python (pandas, matplotlib, seaborn, plotly, folium - Map, Choropleth, Marker, MarkerCluster)

05 - Построение дашбордов в Tableau. Цель проекта - создание дашборда на основе данных о конференциях TED. Инструменты: Tableau

06 - Определение текущего уровня потребительской лояльности. Цель - определить текущий уровень потребительской лояльности, или NPS (от англ. Net Promoter Score), среди клиентов из России. Инструменты: Python (pandas, numpy), SQL, Tableau.

07 - Анализ бизнес‑показателей приложения PROCRASTINATE PRO+. Цель исследования: разобраться в причинах убытков и помочь компании выйти в плюс. Несмотря на огромные вложения в рекламу, последние несколько месяцев компания терпит убытки. Инструменты: Python (pandas, numpy, matplotlib). 

08 - Анализ данных вакансий HH.ru. Цель проекта: Выявить различия в предлагаемых вакансиях для Аналитиков данных и Системных аналитиков. Инструменты: Python (pandas, matplotlib, seaborn)

09 - Статистический анализ данных сервиса аренды самокатов GoFast. Цель исследования - анализ данных и проверка гипотез, которые могут помочь бизнесу вырасти. Инструменты: Python (pandas, matplotlib, seaborn, scipy)

10 - Приоритизация гипотез, A/B‑тестирование. Цель исследования: Приоритизация гипотез для увеличения выручки, запуск A/B-теста и анализ результатов. Инструменты: Python (pandas, matplotlib, seaborn, scipy)

11 - Стратегия взаимодействия с клиентами фитнес‑центра. Цель исследования — провести анализ и подготовить план действий по удержанию клиентов. А именно:
•	прогноз вероятности оттока (на уровне следующего месяца) для каждого клиента;
•	формирование типичных портретов клиентов: выделение нескольких наиболее ярких групп и характеритика их основных свойств;
•	анализ основных признаков, наиболее сильно влияющих на отток;
•	формулирование основных выводов и разработка рекомендаций по повышению качества работы с клиентами
Инструменты: Python (pandas, matplotlib, seaborn, plotly, scipy, sklearn)

12 - Выявление закономерности успешности компьютерной игры. Цель выявление определяющих успешность игры закономерности. Это позволит сделать ставку на потенциально популярный продукт и спланировать рекламные кампании. 
Инструменты: Python (pandas, matplotlib, seaborn, scipy)

13 - Анализ оттока клиентов. Цель исследования - анализ клиентов регионального банка и выделение сегментов клиентов, которые склонны уходить из банка. Инструменты: Python (pandas, matplotlib, seaborn, scipy)
